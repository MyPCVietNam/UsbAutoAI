# Hướng dẫn cài đặt

UsbAutoAI được thiết kế để người không chuyên vẫn có thể làm theo quy trình cơ bản.

> Trước khi cài Windows, hãy sao lưu dữ liệu quan trọng. Việc cài đặt hệ điều hành có thể làm mất dữ liệu nếu chọn nhầm ổ đĩa hoặc phân vùng.

## Quy trình cơ bản

1. **Cắm USB**

   Cắm USB UsbAutoAI vào máy cần cài Windows.

2. **Boot máy**

   Khởi động máy và chọn boot từ USB. Phím boot tùy dòng máy có thể khác nhau, thường là F12, F11, F9, ESC hoặc DEL.

3. **Chọn bộ Windows**

   Chọn phiên bản Windows phù hợp, ví dụ Windows by MyPC hoặc Windows Microsoft Original.

4. **Chờ hệ thống tự động hoàn tất**

   Sau khi xác nhận lựa chọn, hệ thống sẽ tự động xử lý các bước cài đặt còn lại. Người dùng chỉ cần chờ đến khi máy vào desktop.

## Lưu ý khi sử dụng

- Kiểm tra đúng máy và đúng ổ đĩa trước khi cài.
- Nên cắm sạc với laptop trong suốt quá trình cài đặt.
- Không rút USB hoặc tắt máy khi hệ thống đang thao tác.
- Sau khi vào desktop, có thể kết nối internet để Windows Update cập nhật thêm driver nếu cần.

UsbAutoAI giúp đơn giản hóa quá trình cài đặt, nhưng người dùng vẫn nên kiểm tra dữ liệu và lựa chọn phiên bản Windows phù hợp trước khi bắt đầu.

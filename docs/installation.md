# Hướng dẫn cài đặt

UsbAutoAI được thiết kế để quy trình cài Windows dễ hiểu hơn, kể cả với người không chuyên.

> Trước khi cài Windows, hãy sao lưu dữ liệu quan trọng. Cài đặt hệ điều hành có thể làm mất dữ liệu nếu chọn sai ổ đĩa hoặc phân vùng.

## Quy trình cơ bản

1. **Cắm USB**

   Cắm USB UsbAutoAI vào máy cần cài Windows.

2. **Boot máy từ USB**

   Khởi động máy và mở menu boot. Phím boot tùy dòng máy, thường là F12, F11, F9, ESC hoặc DEL.

3. **Chọn bộ Windows**

   Chọn phiên bản phù hợp, ví dụ Windows by MyPC hoặc Windows Microsoft Original.

4. **Chờ hệ thống hoàn tất**

   Sau khi xác nhận, hệ thống sẽ tự động xử lý quy trình cài đặt. Người dùng chỉ cần chờ đến khi máy vào desktop.

## Lưu ý quan trọng

- Kiểm tra đúng máy, đúng ổ đĩa và dữ liệu cần giữ trước khi bắt đầu.
- Với laptop, nên cắm sạc trong suốt quá trình cài đặt.
- Không rút USB hoặc tắt máy khi hệ thống đang thao tác.
- Sau khi vào desktop, nên kết nối internet để Windows Update kiểm tra cập nhật và driver nếu cần.

UsbAutoAI giúp giảm thao tác thủ công, nhưng người dùng vẫn nên kiểm tra kỹ dữ liệu và lựa chọn phiên bản Windows phù hợp trước khi cài.

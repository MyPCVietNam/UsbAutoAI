# Changelog

Trang này ghi nhận các thay đổi quan trọng của tài liệu và định hướng sản phẩm.

## 2026

### Initial GitBook structure

- Tạo cấu trúc GitBook cho UsbAutoAI.
- Thiết lập README.md làm trang chủ sản phẩm.
- Bổ sung SUMMARY.md để điều hướng tài liệu.

### Clean Windows positioning

- Định vị UsbAutoAI là nền tảng cài đặt Windows tự động hiện đại.
- Nhấn mạnh Windows sạch, ổn định, tương thích và dễ bảo trì.
- Làm rõ UsbAutoAI không phải Ghost Windows, công cụ crack hoặc Windows Lite/mod sâu.

### Copyright-safe positioning

- Bổ sung thông điệp không thay thế license Windows, Office hoặc phần mềm bên thứ ba.
- Làm rõ người dùng và doanh nghiệp cần sử dụng giấy phép hợp lệ.
- Tránh ngôn ngữ có thể gây hiểu nhầm về kích hoạt trái phép.

### Added professional documentation pages

- Thêm trang tuân thủ bản quyền.
- Thêm chính sách giấy phép phần mềm.
- Thêm lưu ý an toàn và miễn trừ trách nhiệm.

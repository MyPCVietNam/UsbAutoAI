# Quy trình cho kỹ thuật viên

Trang này gợi ý quy trình thực tế cho kỹ thuật viên khi sử dụng UsbAutoAI để cài Windows cho khách hàng hoặc nhiều thiết bị.

## 1. Checklist trước khi cài

- Xác nhận yêu cầu của khách hàng.
- Kiểm tra model máy, CPU, RAM, ổ cứng và chuẩn boot.
- Kiểm tra tình trạng pin hoặc nguồn điện.
- Hỏi rõ dữ liệu cần giữ.
- Xác định phiên bản Windows phù hợp.
- Xác nhận license Windows và phần mềm cần dùng.

## 2. Checklist sao lưu dữ liệu

- Desktop.
- Documents.
- Downloads.
- Pictures, Videos, Music.
- Dữ liệu phần mềm kế toán, bán hàng hoặc phần mềm chuyên dụng.
- Dữ liệu trình duyệt nếu khách yêu cầu.
- Key, license hoặc tài khoản quan trọng.

> Không bắt đầu cài đặt nếu chưa rõ dữ liệu nào cần giữ.

## 3. Kiểm tra BIOS/UEFI

- Xác định phím boot của máy.
- Kiểm tra chế độ UEFI hoặc Legacy nếu cần.
- Kiểm tra Secure Boot theo yêu cầu của phiên bản Windows.
- Đảm bảo máy nhận USB UsbAutoAI.

## 4. Chọn phiên bản Windows

- Chọn Windows 10 nếu máy cũ hoặc cần tương thích rộng.
- Chọn Windows 11 nếu phần cứng phù hợp và người dùng cần trải nghiệm mới hơn.
- Chọn Microsoft Original nếu khách cần môi trường gần mặc định Microsoft.
- Chọn Windows by MyPC nếu khách cần trải nghiệm sạch, gọn và dễ dùng hơn sau cài.

## 5. Tiến hành cài đặt

- Boot vào USB UsbAutoAI.
- Chọn đúng bộ Windows.
- Kiểm tra lại ổ đĩa trước khi xác nhận.
- Theo dõi quá trình cài đặt.
- Không rút USB hoặc tắt máy khi hệ thống đang thao tác.

## 6. Checklist sau cài đặt

- Kiểm tra máy vào desktop ổn định.
- Kiểm tra Windows Update.
- Kiểm tra Windows Defender.
- Kiểm tra driver mạng, âm thanh, đồ họa, touchpad và các thiết bị chính.
- Kiểm tra ngày giờ, ngôn ngữ và bàn phím.
- Cài ứng dụng cần thiết từ nguồn hợp lệ.

## 7. Kiểm tra license

- Kiểm tra trạng thái kích hoạt Windows bằng phương thức hợp lệ.
- Không dùng crack, bypass hoặc key không rõ nguồn gốc.
- Nếu khách có license, hỗ trợ nhập key hoặc đăng nhập tài khoản phù hợp.
- Với doanh nghiệp, tuân thủ chính sách OEM, Retail, Volume hoặc Digital License.

## 8. Bàn giao cho khách hàng

- Thông báo phiên bản Windows đã cài.
- Nhắc khách kiểm tra dữ liệu đã sao lưu.
- Hướng dẫn cập nhật Windows khi cần.
- Thông báo những phần mềm chưa cài nếu thiếu license hoặc nguồn cài hợp lệ.
- Ghi nhận lỗi phần cứng hoặc driver nếu có.

Một quy trình bàn giao rõ ràng giúp giảm hiểu nhầm và tăng tính chuyên nghiệp khi hỗ trợ khách hàng.
